# How to get D.eSystem

Go to tags and choose a version of D.eSystem.

# Hardware requirements
- a 64 bit CPU
- a UEFI firmware

# About this repo

Here is every D.eSystem version since D.eSystem 6.0.6 alpha, this will be the repo for every version of D.eSystem which will release in the future.
